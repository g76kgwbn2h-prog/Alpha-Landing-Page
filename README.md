# Alpha Landing Page Clone

This project is a static clone of `https://www.neighbourhood.au/alpha` prepared for GitHub hosting.

## Edit event details

1. Open `/Users/kierancarr/Downloads/Alpha Landing Page/event-config.js`.
2. Update text fields, date/location, and the RSVP link.
3. Update image paths in `images` to swap page photos.
3. Save and deploy.

## Files

- `/Users/kierancarr/Downloads/Alpha Landing Page/index.html`: page markup/styles.
- `/Users/kierancarr/Downloads/Alpha Landing Page/event-config.js`: all editable event details.
- `/Users/kierancarr/Downloads/Alpha Landing Page/event-overrides.js`: applies config values to the page.

## Deploy on GitHub Pages

1. Push these files to a GitHub repo.
2. In repo settings, enable GitHub Pages for the branch/folder containing `index.html`.
